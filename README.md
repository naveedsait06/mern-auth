# 🛡️ Secure Systems - MERN Auth

A premium full-stack authentication system built with the MERN stack. Features a modern Bento-grid dashboard, automated email OTP verification, and secure JWT-based session management.

## 🚀 Key Features
*   **Modern Auth Flow**: Secure Login, Registration, and Password Reset.
*   **Email Verification**: Real-time OTP delivery via Brevo (Sendinblue).
*   **Bento UI**: High-end, responsive dashboard design using Tailwind CSS.
*   **Route Protection**: Advanced logic to prevent unverified access to the dashboard.

## 🛠️ Tech Stack
*   **Frontend**: React.js, Tailwind CSS, React Router, Axios.
*   **Backend**: Node.js, Express.js, MongoDB, Nodemailer.
*   **Security**: JWT (JSON Web Tokens), bcrypt.js for password hashing.

## ⚙️ Setup Instructions
1. Clone the repository.
2. Install dependencies in both `/client` and `/server` using `npm install`.
3. Set up your `.env` variables (refer to `.env.example`).
4. Run the backend: `npm run server`.
5. Run the frontend: `npm run dev`.
